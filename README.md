# kue-watch-on-3000

a kue web service running on port 3000

## Install
Install the module with:

```bash
npm install kue-watch-on-3000
```

## Usage
```javascript
var kue_watch_on_3000 = require('kue-watch-on-3000');
kue_watch_on_3000.awesome(); // "awesome"
```
